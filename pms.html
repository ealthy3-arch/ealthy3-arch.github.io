Features

현장에서 가장 많이 쓰는 기능만
크게, 단순하게 배치했습니다

화려한 기능보다 실수 줄이기와 복구 쉬운 흐름을 먼저 설계했습니다.

객실 카드 상태관리

객실 번호 카드 색상과 텍스트를 같이 보여줘 현재 상태를 바로 확인할 수 있습니다.

예약 → 체크인 전환

예약 목록에서 객실을 선택해 즉시 체크인으로 바꿀 수 있어 중복 입력을 줄입니다.

체크아웃/노쇼/환불

상태별 버튼을 분리해 눌러야 할 동작을 명확하게 안내합니다.

일마감 정산

현금/카드/환불 내역을 하루 단위로 모아 정산표와 CSV를 동시에 확인합니다.

로그와 추적

중요 작업은 사용자 ID와 날짜/시간 로그를 남겨 나중에 확인할 수 있습니다.

로컬 우선 + 백업

기본은 로컬 DB로 안전하게 운영하고, 필요할 때만 동기화 설정을 적용합니다.

Workflow

매일 같은 순서로 운영할 수 있게
단계를 고정했습니다

1. 예약 확인

당일 예약 목록에서 입실 예정 고객을 먼저 확인합니다.

2. 체크인/객실 배치

예약자를 선택해 객실 카드를 지정하고 체크인 처리합니다.

3. 체크아웃/정산 마감

퇴실 처리 후 일마감 정산으로 하루 데이터를 저장합니다.
